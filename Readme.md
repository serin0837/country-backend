# Description

Back end made with Node, Express, MongoDB.

# API End point

- /countries -> get, post
- /imaginarycountries -> get, post, delete, patch(update)
- /countries/(countryname)-> get
- /countries/(countryid)-> update, delete
- /region/(regionname)-> get
